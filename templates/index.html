Traffic Density Classification

This project is web application version of my undergraduate research titled Traffic Density Classification Based on Texture Descriptor. Instead of using four texture descriptors which are RGB, grayscale, image gradient, and local binary pattern, the texture descriptor used in this project is only local binary pattern.
This web application was built with mainly Python (Flask and Keras Tensorflow), CSS, and HTML. There are a lot of things that can be fixed and improved from this web application.
